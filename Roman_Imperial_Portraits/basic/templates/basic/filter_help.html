Open the filter tab

Most listviews will have a 'filter' tab. Search the top-right area of the screen for this icon: .
Press the filter icon once to open the filter 'tab' (area to specify filters), and press it again to close that tab.

Select one or more filters

The filter tab shows a row of 'selected filters'.
To actually use one of these filters, press it, so that its color changes to green.

The tab may also contain one or more filters that open up a new row of filters.
These 'expansion' filters can be recognized by the three dots '...' at the end of the filter name.
For example: Collection...

The filters come in a limited number of types:

  • Text: Either provide the exact text of what is being looked for (e.g. author Augustinus Hipponensis (pseudo)), or else use the wildcard * to indicate what you mean (e.g. author *august*). It is also possible to use a # mark to single out one word or string (e.g. hipponensis#)
  • A number: Provide an integer (whole) number, i.e. one without dot or comma in it.
  • Item selection: Select one or more items from the options provided. Start typing and options will appear. Select an option by either clicking it with the mouse or by pressing the Enter key.

Use the Search button

Once all criteria have been set, use the button Search to start searching. The results will be provided in one or more pages (depending on how many results there are). Go to a particular page by simply clicking the page number.

Use the button Clear to clear the search criteria. Note that it is still necessary to press Search in order to get the application to provide the complete listview.
